# 🏏 IPL Match Winner Predictor

IPL is chaos… but I tried predicting it anyway 😏🔥

This Machine Learning model predicts IPL match winners using:
- 📊 10+ years of historical data
- 🏟️ Home ground advantage
- 📈 Team form & stats

⚡ Accuracy: ~57% — because IPL is unpredictable!
